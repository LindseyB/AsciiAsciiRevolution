Description:
This module contains a set of constant error codes used by parts of the ncurses package.

E_OK
Success

E_BAD_ARGUMENT
Incorrect/out of range argument

E_BAD_STATE
Routine called from initialization or termination function

E_NO_MATCH
Character match fail

E_NO_ROOM
Menu too large for window

E_NOT_CONNECTED
No items in menu

E_NOT_POSTED
Menu hasn't been posted

E_NOT_SELECTABLE
Item can't be selected

E_POSTED
Menu already posted

E_REQUEST_DENIED
Menu driver couldn't process request

E_SYSTEM_ERROR
System error

E_UNKNOWN_COMMAND
Menu driver saw an unknown request code

E_CONNECTED
E_INVALID_FIELD
E_CURRENT
undocumented
